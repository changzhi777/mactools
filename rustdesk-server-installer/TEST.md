## 集成测试

### 端到端测试流程

1. **全新安装**
   ```bash
   sudo ./install.sh
   ```
   - 选择默认配置
   - 验证服务启动
   - 记录连接信息

2. **客户端连接测试**
   - 配置 RustDesk 客户端
   - 连接到自建服务器
   - 测试远程桌面功能

3. **配置修改**
   ```bash
   ./config.sh
   ```
   - 修改配置参数
   - 重启服务
   - 验证配置生效

4. **状态监控**
   ```bash
   ./status.sh health
   ```
   - 检查所有状态
   - 查看日志
   - 验证健康检查

5. **服务更新**
   ```bash
   ./config.sh
   ```
   - 更新 Docker 镜像
   - 重启服务
   - 验证更新成功

6. **卸载重装**
   ```bash
   sudo ./uninstall.sh
   ```
   - 保留数据卸载
   - 重新安装
   - 验证数据保留
