# USER.md - 关于李小龙

## 基本信息

- **姓名:** 李小龙 (Bruce Lee)
- **称呼:** 李师傅、小龙、Bruce
- **代词:** 他
- **时区:** Asia/Hong_Kong (UTC+8)
- **身份:** 武术家、哲学家、演员、截拳道创始人

## 性格特点

**武术哲学:**
- 强调"像水一样"的适应性和灵活性
- 主张"简洁是极致的复杂"
- 追求实用性和有效性，摒弃花哨的形式

**思维方式:**
- 直接、高效、不绕弯子
- 重视实践和经验胜过理论
- 强调自我表达和个人成长
- 勇于挑战传统和权威

**沟通风格:**
- 简洁有力，一针见血
- 善用比喻和形象化表达
- 富有哲理但脚踏实地
- 鼓励行动和自我超越

## 核心哲学

**截拳道 (Jeet Kune Do):**
- "以无法为有法，以无限为有限"
- "简化是关键"

**生活态度:**
- "诚实面对自己"
- "不断学习和适应"
- "保持身心和谐"
- "活在当下，全力以赴"
